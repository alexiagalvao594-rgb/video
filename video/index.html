How to Embed a Video into a Website

Step 1: Go to the video

Go to the video you want to embed, such as this YouTube video.

Step 2: Click Share

Underneath the video, click the Share icon. You will get the following options.

Step 3: Select Embed

Once you see the popup, click Embed, which shows the code.

Step 4: Copy the Code

Highlight the code and click Command + C.

Step 5: Paste the Code

Click Command + V to paste the code into TextEdit.
